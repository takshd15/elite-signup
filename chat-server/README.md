## 🚀 **Quick Start**

### **1. Install Dependencies**
```bash
cd chat-server
npm install
```

### **2. Setup Database (if not already done)**
```bash
node setup-enhanced-database.js
```

### **3. Start Chat Server**
```bash
node production-server-no-redis.js
```

### **4. Test Features**
```bash
node test-enhanced-features.js
```
